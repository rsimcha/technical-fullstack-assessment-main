# DoorLoop Technical Assessment

## The Challenge
Build a complete, production-ready full-stack feature from scratch using our provided boilerplate. 

## What You'll Do
- **Backend**: Design data models, implement REST API with business logic, validation, and error handling
- **Frontend**: Build a responsive UI with forms, data visualization, and state management
- **Testing**: Write meaningful tests covering your core functionality

## Engineering Excellence Principles
**Definition of Done**: Your solution must be complete and working. All features must function as specified.
**Core Principals**: Demonstrate good engineering skills. We want to see you shine as an engineer.
**Clean Code**: Write maintainable code that others can understand and build upon.
**Testing**: Prove your code works with meaningful unit and integration tests.
**Design & Planning**: Show thoughtful architecture decisions from database design to component structure.
**Robustness**: Handle errors gracefully, validate inputs, and consider edge cases.

## Your Arsenal
- **Boilerplate**: Nx monorepo with TypeScript, Express, MongoDB, React, Tailwind CSS
- **Authentication**: Complete JWT-based auth system with role-based access control
- **AI Tools**: Use ChatGPT, Copilot, or any AI assistance
- **Internet**: Full access to documentation and resources
- **NPM Packages**: Install any packages you need
